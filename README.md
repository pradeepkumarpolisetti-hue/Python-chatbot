# Python Chatbot

This is a simple rule-based chatbot built using Python.

## Features
- Handles greetings like "hi" and "hello"
- Responds to keywords like "AI"
- Accepts user name
- Continuous conversation using loop
- Exit command to stop chatbot

## How it Works
The chatbot takes user input, processes it using conditional logic, and responds accordingly.

## What I Learned
- Conditional statements (if-else)
- Loops for continuous interaction
- Handling user input
- Building logic step by step

## Future Improvements
- Add memory for conversations
- Improve responses
- Integrate AI APIs

## How to Run
1. Run chatbot.py
2. Start chatting in terminal
